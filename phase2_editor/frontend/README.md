# Editor Frontend

React wizard frontend for the Phase 2 editor.

This is the active imported editor interface. It includes:

- Native folder Browse flow through the backend.
- Autosave resume prompt.
- Five-stage wizard navigation.
- Field visibility, label, and order editing.
- Spreadsheet-style feature editing table.
- Reference map panel for the selected layer/row.
- Search, edited-only, and errors-only filters.
- Undo, field revert, and row revert actions.
- Validation issue review.
- Preview and export controls.
