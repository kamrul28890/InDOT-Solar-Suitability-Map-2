# INDOT Solar Suitability Workspace

This repository is now organized into two project areas:

- `phase1_map/` - the completed non-editor INDOT Solar Suitability Map, including the existing FastAPI API, React/Leaflet frontend, data export scripts, tests, release assets, and Windows package artifacts.
- `phase2_editor/` - the new editor-mode application work, starting from the Phase 2 implementation plan.

Use `phase1_map/` when running or rebuilding the existing map package. Use `phase2_editor/` for all new editor implementation.
