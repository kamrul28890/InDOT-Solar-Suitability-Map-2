# Legacy Tools Archive

This folder holds standalone tools that are no longer the active public website source.

## `indot-shapefile-editor/`

Archived from `D:\My Projects\InDOT-Shapefile-Editor`.

This was the earlier Windows/local shapefile editor for importing, editing, validating, and exporting INDOT solar map data packages. The current public website source lives in `website/`, and the integrated browser-based Map Builder is maintained there.

Keep this archived copy only as a rollback/reference for the older standalone editor workflow.
